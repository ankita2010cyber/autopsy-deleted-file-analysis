# Forensic Analysis of Deleted Files Using Autopsy

## Objective

This project demonstrates beginner-level Digital Forensics and Incident Response (DFIR) techniques using Autopsy for deleted file investigation and NTFS artifact analysis.

## Tools Used

* Autopsy
* The Sleuth Kit
* Windows

## Investigation Performed

* Disk image examination
* Deleted file investigation
* NTFS file system analysis
* Metadata examination
* File recovery analysis

## Findings

The investigation identified deleted file artifacts and analyzed file recovery limitations caused by overwritten or missing data blocks within the NTFS file system.

## Skills Gained

* Digital Forensics
* File Recovery Analysis
* NTFS Artifact Investigation
* Evidence Examination
* DFIR Fundamentals
